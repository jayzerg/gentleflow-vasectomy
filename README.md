# GentleFlow | Modern No-Scalpel Vasectomy Care

A premium, 2026-standard healthcare booking platform focused on trust, transparency, and clinical excellence. Built with a Neumorphic "Stone & Gold" aesthetic to provide a calm, high-end user experience for patients seeking surgical freedom.

## 🌟 Purpose

GentleFlow aims to reduce patient anxiety and streamline the booking process for no-scalpel vasectomy procedures. By combining modern UI/UX with practical medical tools, we bridge the gap between clinical necessity and patient comfort.

## 🚀 Features

### Patient Engagement
- **Neumorphic Design:** Soft UI (Stone & Gold palette) for a premium, trustworthy medical feel.
- **GentleBot Chat:** Instant automated answers for common procedure and pricing queries.
- **Scroll Reveal Animations:** Interactive components that engage patients as they explore.

### Clinical Clarity
- **The Journey:** Visual 5-stage timeline from Discovery to Clearance.
- **Detailed FAQ:** Comprehensive accordion addressing medical, financial, and personal concerns.
- **Surgeon Portfolio:** Verified credentials and gallery of modern clinical standards.

### Transactional Tools
- **Insurance Estimator:** Real-time out-of-pocket cost calculator based on deductibles and co-pays.
- **Live Availability:** Interactive calendar slot selection for expedited booking.
- **Secure Requests:** Direct booking form with discrete data handling simulation.

## 🛠️ Setup Instructions

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jayzerg/gentleflow-vasectomy.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gentleflow-vasectomy
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the local development server:
```bash
npm run dev
```
The site will be available at `http://localhost:5173/`.

### Production Build
Generate optimized production assets:
```bash
npm run build
```

## 📖 Usage Guidelines

### Navigation
- Use the **Fast Track Booking** button in the header for immediate access to scheduling.
- The **Journey** section provides a step-by-step walkthrough for first-time patients.
- Use the **Insurance Estimator** before booking to get a clear financial picture.

### Chatbot
- Click the floating **💬 message bubble** for quick answers.
- Keywords: `price`, `cost`, `slots`, `book`.

## 🔒 Confidentiality
GentleFlow is designed with privacy-first principles. In a production environment, all data is encrypted via end-to-end protocols.
